Welcome to SignelJS

A reactive UI library to make your DOM alive, simple, and powerful.

Explore Features
Reactive Arrays Automatically update your lists when data changes. No extra DOM manipulation needed. 

// Example: Reactive array in SignelJS
const todoList = new El("#todoList", proxy => {
  proxy.list("todos", ["Buy milk", "Learn JS", "Build Resign.js demo"], item => `<li>${item}</li>`)
});

// Adding new element in list
new El("#addTodo", proxy => {
  proxy.on("click", () => {
    var newItem = document.querySelector("#newItem").value;
    todoList.todos.push(newItem);
  });
});
Reactive Tabs Switch between tabs easily. Only one active at a time, fully reactive UI. 

// Example: Reactive tabs in SignelJS
const tabs = new El("#tab-buttons", proxy => {
  proxy.tab("tab1", "#tab1");
  proxy.tab("tab2", "#tab2");
  proxy.tab("tab3", "#tab3");
});

// Switching tabs when clicked
El("#tab1-btn", proxy => {
  proxy.on("click", ()=> tabs.tab1.active = !tabs.tab1.active);
});

El("#tab2-btn", proxy => {
  proxy.on("click", ()=> tabs.tab2.active = !tabs.tab2.active);
});

El("#tab3-btn", proxy => {
  proxy.on("click", ()=> tabs.tab3.active = !tabs.tab3.active);
});
